The following are the different tags of HTML-5

1. doctype: It is used to specify the version of HTML.

2. html: It is used to define the root of an HTML document.

3. head: It is used to define the head of an HTML document.

4. title: It is used to define the title of an HTML document.

5. body: It is used to define the body of an HTML document.

6. h1 to h6: It is used to define the headings of an HTML document.

7. p: It is used to define a paragraph of an HTML document.

8. br: It is used to define a line break of an HTML document.

9. hr: It is used to define a thematic break of an HTML document.

10. a: It is used to define a hyperlink of an HTML document.

11. img: It is used to define an image of an HTML document.

12. ul: It is used to define an unordered list of an HTML document.

13. ol: It is used to define an ordered list of an HTML document.

14. li: It is used to define a list item of an HTML document.

15. table: It is used to define a table of an HTML document.

16. tr: It is used to define a table row of an HTML document.
